선언적 렌더링

  • DOM에서 데이터를 선언적으로 렌더링 할 수 있음.
  • 데이터와 DOM은 연결되어 있으며, 모든 건 반응형!
  • app.message를 다른 값으로 설정하면 그에 따라 자동으로 값이 업데이트 됨.

    • Ex

    {{ message }}

    디렉티브

  • Vue에서 제공하는 특수 속성
  • v- 접두어가 붙어있음

    • Ex

    바인딩

    v-bind:title="message" : 해당 요소의 title 속성을 Vue 인스턴스의 message값으로 최신 상태를 유지함.

    이 위에 마우스를 올리면 동적으로 바인딩 된 title을 볼 수 있습니다.

    v-if="seen" :

    app-3의 seen값이 true라면 이 글을 볼 수 있습니다. 콘솔에 app3.seen = false; 를 입력하면 글이 사라집니다.
